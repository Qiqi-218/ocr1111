# 局域网 OCR 网站（前后端一体）

本项目提供在局域网可访问的图片文字识别网站：
- 前端：图片上传、任意角度旋转、框选裁剪、四点透视、自动矫正、本地OCR与AI识别、TXT/DOCX导出。
- 后端：Node/Express 静态服务与 AI 转发 API（固定调用火山引擎 Ark）。

## 启动
1. 安装依赖
```
npm install
```
2. 配置环境变量（在项目根目录创建 `.env` 文件，拷贝并修改如下内容）：
```
HOST=0.0.0.0
PORT=5173
VOLC_BASE_URL=https://ark.cn-beijing.volces.com/api/v3
VOLC_API_KEY=替换为你的火山引擎APIKey
VOLC_MODEL=doubao-seed-1-6-vision-250815
```
3. 启动服务
```
npm start
```
在局域网其他设备通过 `http://<本机IP>:5173` 访问。

## 使用要点
- 自动矫正：点击“自动矫正”按钮，基于边缘/轮廓检测自动寻找四边形文档并矫正。
- 框选裁剪：支持任意旋转/翻转/缩放；内部将画布选择区域逆变换到原图再透视裁剪。
- 四点矫正：点击四个角点后“应用裁剪/矫正”。
- AI 识别：无需在前端填写密钥，后端统一代理到火山引擎 Ark。

## 常见问题
- OpenCV.js 首次加载需要时间；状态栏显示“OpenCV.js 已就绪”后再使用矫正功能。
- 若 AI 接口 401/403，请确认 `.env` 的 `VOLC_API_KEY` 与权限。
- 若跨设备无法访问，确认防火墙放行端口（默认 5173），以及服务监听 `HOST=0.0.0.0`。
